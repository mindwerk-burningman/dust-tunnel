# DustTunnel #

## Requirements ##

* oscP5
* netP5 (comes with oscP5?)
* themidibus

## Installation ##

* transfer DustTunnel
* install intelliJ
* install/transfer requirements to Documents > Processing > libraries
* sign into app store and install Logic

## Setup ##

* set up an IAC bus via "Audio MIDI Setup" > show midi environment
* restart logic
